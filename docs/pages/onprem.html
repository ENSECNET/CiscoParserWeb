On-prem edition

When you want a live NetBox push instead of a download.

The CiscoParser repo is the on-prem edition: FastAPI + Docker on a Proxmox LXC, pushing parsed devices straight into NetBox over your local network.

.cfg ──► parse ──► transform ──► netbox_push ──► NetBox

Same parser logic, same model shape — the on-prem edition adds the live push and a NetBox connection test. Use the web edition for quick conversions, the on-prem edition for routine documentation against your own NetBox.
