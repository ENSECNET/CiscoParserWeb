Deploy on Cloudflare

The standard ENSECNET workflow: GitHub repo → Cloudflare import → auto-deploy.

  1. Upload this repo to GitHub (ensecnet/CiscoParserWeb).
  2. Cloudflare → Workers & Pages → Import a repository → select the repo.
  3. Deploy command: npx wrangler deploy. Build command: empty.
  4. wrangler.jsonc binds the custom domain cisco.ensecnet.net.

Push to main → Cloudflare auto-deploys. No build step, no dependencies — the Worker is plain JavaScript.

Nasadenie na Cloudflare

Štandardný ENSECNET workflow: GitHub repo → import do Cloudflare → auto-deploy.

  1. Nahraj toto repo na GitHub (ensecnet/CiscoParserWeb).
  2. Cloudflare → Workers & Pages → Import a repository → vyber repo.
  3. Deploy command: npx wrangler deploy. Build command: prázdne.
  4. wrangler.jsonc nabinduje custom doménu cisco.ensecnet.net.

Push na main → Cloudflare auto-deployne. Žiadny build krok, žiadne závislosti — Worker je čisté JavaScript.
