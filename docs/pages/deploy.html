Deploy on Cloudflare

The standard ENSECNET workflow: GitHub repo → Cloudflare import → auto-deploy.

  1. Upload this repo to GitHub (ensecnet/CiscoParserWeb).
  2. Cloudflare → Workers & Pages → Import a repository → select the repo.
  3. Deploy command: npx wrangler deploy. Build command: empty.
  4. wrangler.jsonc binds the custom domain cisco.ensecnet.net.

Push to main → Cloudflare auto-deploys. No build step, no dependencies — the Worker is plain JavaScript.
