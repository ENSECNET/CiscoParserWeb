CiscoParser WebENSECNET · public converter

How it works

A pure-JavaScript parser running on the Cloudflare edge — no server, no database.

Endpoints

MethodPathPurpose
GET/the converter UI
POST/api/parse{config} → NetBox model JSON
POST/api/snippet{model} → import.py text

Output

  • Model JSON — drop into automation, diff against last time, or archive.
  • import.py — a self-contained pynetbox script you run on a host that can reach your own NetBox, with your own NETBOX_URL / NETBOX_TOKEN.

Parser parity

The JS parser produces the same model shape as the Python on-prem edition (name, platform, interfaces[], vlans[], vrfs[], config_context{}), so the JSON and the generated script are interchangeable with the on-prem tool.
