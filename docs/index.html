CiscoParser Web

A public, zero-install converter — turn a Cisco running-config into a NetBox-ready model, in your browser.

Cloudflare Worker pure JavaScript nothing stored
Paste a Cisco IOS-XE running-config and get a structured NetBox model — interfaces, IPs, VLANs, VRFs, plus routing / AAA / SNMP captured as config context. Everything runs in your browser session. No backend database, no credentials leave your machine.

How it works

paste config ──► parse (Worker, JS) ──► NetBox model JSON ├─► Download JSON └─► Download import.py (run it yourself)

Because NetBox lives on a private network and a token is sensitive, the web edition does not push. It returns the model as JSON and a self-contained import.py you run against your own NetBox. See Privacy.

CiscoParser Web

Verejný konverter bez inštalácie — premeň Cisco running-config na model pripravený pre NetBox, priamo v prehliadači.

Cloudflare Worker čisté JavaScript nič sa neukladá
Vlož Cisco IOS-XE running-config a dostaneš štruktúrovaný NetBox model — rozhrania, IP, VLAN-y, VRF-y, plus routing / AAA / SNMP zachytené ako config context. Všetko beží v relácii tvojho prehliadača. Žiadna backend databáza, žiadne prihlasovacie údaje neopustia tvoj stroj.

Ako to funguje

vlož config ──► parse (Worker, JS) ──► NetBox model JSON ├─► Stiahni JSON └─► Stiahni import.py (spustíš sám)

Keďže NetBox žije v privátnej sieti a token je citlivý, webová edícia nepushuje. Vráti model ako JSON a samostatný import.py, ktorý spustíš proti svojmu vlastnému NetBoxu. Pozri Súkromie.
