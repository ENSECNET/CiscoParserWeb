CiscoParser Web

A public, zero-install converter — turn a Cisco running-config into a NetBox-ready model, in your browser.

Cloudflare Worker pure JavaScript nothing stored
Paste a Cisco IOS-XE running-config and get a structured NetBox model — interfaces, IPs, VLANs, VRFs, plus routing / AAA / SNMP captured as config context. Everything runs in your browser session. No backend database, no credentials leave your machine.

How it works

paste config ──► parse (Worker, JS) ──► NetBox model JSON ├─► Download JSON └─► Download import.py (run it yourself)

Because NetBox lives on a private network and a token is sensitive, the web edition does not push. It returns the model as JSON and a self-contained import.py you run against your own NetBox. See Privacy.
